# Map-files-in-Kelantan
Kelantan State Geospatial Data (Jajahan & Mukim)
This repository provides a comprehensive collection of map files for the state of Kelantan, Malaysia. The data is structured to provide high-granularity access, moving from the state level down to individual administrative districts (Jajahan) and their respective sub-districts (Mukim).

📂 Repository Structure
The data is organized into folders based on the administrative districts of Kelantan. Each folder contains the boundary files for the Mukims within that specific Jajahan.

Map-files-in-Kelantan/
├── Map Kelantan with Mukim/
│   ├── Map GM/           # Gua Musang
│   ├── Map Jeli/         # Jeli
│   ├── Map KK/           # Kuala Krai
│   ├── Map Machang/       # Machang
│   ├── Map P.Mas/        # Pasir Mas
│   ├── Map P.Puteh/      # Pasir Puteh
│   ├── Map TM/           # Tanah Merah
│   ├── Map Tumpat/       # Tumpat
│   ├── Map bck/          # Bachok
│   ├── Map kb/           # Kota Bharu
│   └── Map_kel/          # Kelantan State (Master)
├── LICENSE
└── README.md

 
Navigate to the district: Open the folder corresponding to the area you are interested in (e.g., Map kb for Kota Bharu).Load Data: Import the files into your GIS environment to begin analysis or visualization or test map files by https://mapshaper.org/. 

📌 Administrative Context
In Kelantan, the administrative hierarchy is unique compared to other Malaysian states:State (Negeri): KelantanDistrict (Jajahan): The primary administrative division (e.g., Kota Bharu).Sub-district (Mukim): The localized level of administration covered in this dataset.
